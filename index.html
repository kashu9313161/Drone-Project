What & Why

The Problem MAG-NULL Solves

Modern drones operate in two phases — the second is undetectable by all existing systems. MAG-NULL exploits the only window that exists: the RF active phase.

The Threat
Two-Phase Drone Attack
Phase 1: GPS + active radio datalink to operator. Phase 2: Onboard AI locks target, severs all radio communication. The drone becomes invisible to every RF system on Earth the moment it goes silent.
The Solution
Catch It Before Silence
Every drone must transmit during Phase 1. MAG-NULL passively listens, fingerprints the protocol, assesses threat level, and flags RF silence the moment it occurs — before the kill phase begins.

Python · Zero External Deps

Backend Architecture

Five Python files. Pure stdlib + NumPy + SciPy. No FastAPI, no Flask. Runs anywhere Python runs.

main.py
Entry Point
Starts HTTP server (port 8080), WebSocket server (port 8765), and pipeline tick loop simultaneously using three threads. One command runs everything.
pipeline.py
Master Tick Loop
Runs every 75ms. Wires all DSP stages in sequence. Returns a fully-formed JSON state dict on every tick — spectrum, contacts, swarm score, alerts.
simulator.py
IQ Signal Generator
Generates mathematically accurate IQ bursts for all 4 protocols. Seeded RNG for reproducibility. Gaussian burst shapes, LoRa chirp sweep, OFDM flat spectrum.
dsp.py
Signal Processing
512-point Hanning-windowed FFT. Rolling 12th-percentile noise floor. Energy cluster detection. Feature extraction. Rule-based classifier. Five-factor swarm scorer.
ws_server.py
WebSocket Server
Full RFC 6455 implementation from scratch — SHA1 handshake, frame encoding/decoding, multi-client broadcast, dead connection cleanup. Pure Python stdlib only.

From Simulation → Real Deployment

Hardware Design

One code change converts the simulator to a real HackRF One. Everything else stays identical.

The One Code Change
Simulation → Hardware
# SIMULATION (current)
spectrum = simulator.generate_spectrum(t)

# PRODUCTION — plug in HackRF One
sdr = SoapySDR.Device({'driver':'hackrf'})
sdr.setSampleRate(0, 20e6)
sdr.setCenterFreq(0, 2.44e9)
buff = np.zeros(512, dtype=np.complex64)
sdr.readStream(rx_stream, [buff], 512)
spectrum = compute_spectrum(buff)  # same!

7 Models · 3 Phases

Machine Learning Roadmap

The rule-based classifier works perfectly today. These models upgrade it to probabilistic, robust, and multi-modal intelligence.

Build First · Phase 1
Spectrogram CNN Classifier
MobileNetV2 trained on 128×128 spectrogram images. Your simulator auto-generates perfectly labeled training data. 10,000 images + 20 epochs on CPU = >94% accuracy. Replaces the rule-based classify_protocol() function.
MobileNetV2Auto-labeled data
Build Second · Phase 1
RF Silence Classifier (LSTM)
10-step RSSI sequence → TERMINAL_GUIDANCE vs OUT_OF_RANGE. Positive RSSI slope before silence = approaching target. Negative slope = out of range. Critical for reducing false alarms.
LSTM
Phase 2
Acoustic Motor Classifier
YAMNet fine-tuned on UAV sound dataset. Detects drone motor harmonic frequency signatures using MEMS microphone array. Works after RF silence. ~500m range.
Phase 2
Visual Drone Detector (YOLOv8)
Fine-tuned on Roboflow drone detection dataset. 50+ labeled datasets available. Daylight only, ~300m range. Combined with thermal for 24/7 coverage.
Phase 3
Operator Fingerprinting (Siamese Network)
Matches controller transmission patterns to identify specific operators across multiple incidents. Requires real hardware captures. Long-term intelligence capability.
